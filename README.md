In environments like colleges or hostels where mobile phones are strictly not allowed, students face a major challenge when they need to reach their parents urgently.
I built this platform to provide a simple solution. It is a secure, web-based system where the Admin maps each student to their specific parent using a unique Roll Number. This ensures that even without a mobile phone, a student can easily and safely communicate with their own parents through an authorized web portal.

Live Link: https://emergencycommunication.vercel.app/

Key Highlights:
No Mobile Required: Students can log in and send messages from any computer or shared kiosk.
Verified Mapping: The Admin ensures students are only connected to their registered parents via their Roll Number.
Simple & Fast: Focused strictly on emergency communication to keep it efficient.
